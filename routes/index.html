api usage:

  • POST /api/order - post new order
  • GET /api/orders - get brief info about all orders
  • GET /api/company - get list of companies
  • POST /api/company/:id - get one company by id
  • POST /api/product - get all store products
  • POST /company/:id/product - get all products belonging to a specific company (:id = companyId)
